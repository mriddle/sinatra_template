Sinatra template
==========

Basic Sinatra template setup with:
 * SASS
 * HAML
 * Coffee-script
 * Sprockets 2 (asset pipeline)
 * Underscore & Backbone JS
 * RSpec2
 * ~~MongoDB~~
 * PostgresSQL

 Suggestion: "gem install tux":https://github.com/cldwalker/tux

### Setup

```
cp .ruby-version.example .ruby-version
bundle
````
